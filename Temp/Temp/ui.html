ANT+ Monitor

The application sets the ANT USB stick into a scanning mode searching on the ANT+ frequency 2457Mhz for broadcasts sent by various sensor types, e.g a heart rate monitor.

This project would have been almost impossible without the effort and creativity in supporting dependant libraries and icons.

Libraries

  • Charting Highcharts
  • Data-binding Knockoutjs
  • Script loading Requirejs

Icons

  • Fugue icons Fugue Icons
  • Visual pharm Visual Pharm

The application and the ANT USB library Libant is entierly developed in Javascript by Henning Knut Skoglund, 2013–2014
